Exemplo 8 HTML - Uso das tags <ol>, <ul> e <li>

As tags <&ol>, <&ul> representam listas em html e são utilizadas para as mais diversas finalidades. A <ol> representa uma lista ordenada, já a tag <ul> representa uma lista não ordenada.

Em ambas as listas, para representar os elementos da lista usamos a tag <li>

Além disso, estas listas podem conter outras listas de forma alinhada.

Etapas para desenvolver o site (lista ordenada)

  1. planejamento
    1. definir o objetivo do site
    2. identificar o público-alvo
    3. criar um mapa do site
  2. design
    1. criar wireframes e protótipos
    2. defina a paleta de cores e a tipografia
    3. desenvolver o layout no figma ou adobe xd
  3. desenvolvimento
    1. configurar o ambiente de desenvolvimento
    2. codificar o front-end(html, css, javascript)
    3. implementar o back-end, com linguagens orientadas a objetos (C#, Python php, Java)
  4. testes
    1. testar a responsividade em diferentes aparaelhos
    2. realizar os testes de usabilidade
    3. corrigir bugs identificados
  5. publicação
    1. escolha um provedor da hospedagem
    2. configurar o dominio e o ssl
    3. fazer o deploy do site

Exemplo 9 - Uso das tags de Imagem e Vídeo

Este exemplo mostra diferentes formas de usar as tags <im> e <video>, incluindo o uso de <figure> e <figcaption> para agrupar mídias com legendas.

Descrição

As Tags abordadas são:

  • <img>: Usada para inserir imagens na página.
  • <vido>: Usada para inserir vídeos na página.
  • <figure>: Usada para agrupar mídias (imagens, vídeos, gráficos) com uma legenda.
  • <figcaption>: Usada para adicionar legendas ao conteúdo dentro de <figure>

O que é o Grid Layout

Article

O grid Layout é bidimensional, permitindo que o desnvolvedor possa separar o layout da página em linhas e colunas, diferentemente do flexbox que é unidimensional, ou seja ordena os elementos em linha.

Article

Flexbox ou Grid Layout

O Flexbox e o Grid Layout podem se usados em conjunto. Você pode aplicar o Flexbox a elementos dentro de um Layout que esteja usando o Grid Layout, aproveitando o melhor dos dois mundos.

A escolha entre Flexbox e o Grid Layout não se trata de um substituir o outro, mas sim de entedner as necessidades específicas do seu layout e escolher a técnica que melhor atende a essas necessidades.
